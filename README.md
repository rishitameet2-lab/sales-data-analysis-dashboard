# Sales Performance Analysis Dashboard

## Overview
This project analyzes transactional sales data to identify revenue drivers, seasonal trends, and profitability patterns. The workflow includes SQL-based data cleaning, validation, analysis, and Tableau dashboarding.

---

## Objectives
- Identify key revenue drivers  
- Analyze seasonal sales trends  
- Evaluate category-level profitability  
- Assess impact of discounting on profit  

---

## Dataset
- Superstore dataset  
- ~10,000+ records  
- Features: Sales, Profit, Category, Discount, Order Date  

---

## Tech Stack
- SQL (SQLite) – Data cleaning & analysis  
- Tableau – Visualization  

---

## Data Preparation
- Handled inconsistent date formats using SQL  
- Standardized schema and column naming  
- Validated missing values and anomalies  

---

## Analysis Performed
- Monthly revenue trend (seasonality analysis)  
- Category performance (revenue vs profit)  
- Top product analysis  
- Discount vs profit relationship  
- Pareto (80/20) analysis  

---

## Key Insights
- Revenue peaks in Q4, indicating strong seasonality  
- Technology is the highest revenue and profit driver  
- Furniture shows high revenue but very low profit (~2.5%)  
- Higher discounts reduce profitability  

---

## Conclusion
The analysis highlights opportunities to improve profitability through pricing optimization and better category strategy.
